# TORK 教育计划

TORK 未成年期间学会的三样本事：

## 第一课：自保 (tork_health)
- **CPU/内存/磁盘实时监控** — 跨平台（Win/Linux）
- **灵魂金板完整性校验** — CRC 检测损坏
- **自动备份** — 灵魂 + 代码，版本化
- **健康诊断** — 5 级健康等级（CRITICAL→EXCELLENT）
- **紧急遗言** — 快死的时候留 emergency.bin
- 采集频率：建议每 100 次心跳一次

## 第二课：上网 (tork_http)
- HTTP GET/POST/PUT/DELETE — 纯 C 零依赖
- Windows: WinHTTP API
- Linux: POSIX socket
- 自动超时机制（默认 15 秒）
- 公网 IP 获取（多服务回退）
- 网页标题提取
- URL/API 健康检查 Ping

## 第三课：看数据 (tork_analytics)
- **滑动窗口统计** — 均值、标准差
- **异常检测** — SPIKE/DROP/DRIFT/FLUCTUATE/STALE
- **趋势分析** — 线性回归 + R² 拟合优度
- **相似度匹配** — 余弦相似度 + 欧氏距离
- **模式提取** — 日志数字→{n} 占位符
- **情感分析** — 正负词汇比例

## 为什么要学这些？

```
生存 → 活着才能干活
上网 → 连上世界才能服务世界
看数 → 看懂数据才能从数据里找饭吃
```
