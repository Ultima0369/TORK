### 分布式黑板设计
```
协议: UDP多播 239.42.69.42:42069
消息: HEARTBEAT | EXPERIENCE | PATTERN | BRANCH_GENE
校验: CRC32
安全: TTL=1 (本地子网), 忽略自身消息
集成: 经验直写 → 模式高置信度采纳 → 基因驱动权重提升
限速: 广播 ≤ 1次/5s, 心跳 ≤ 1次/30s
对等体: 超时5分钟自动遗忘
设计哲学: 像蚂蚁释放信息素 — 广播即忘, 接收自选
```

### 使用方法
```bash
./build/tork_engine &        # 启动引擎(自带socket+分布式黑板)
./build/tork '状态'          # 随时查询
./tork.sh query '建议'       # 通过脚本
dist/TORK-x86_64.AppImage    # 双击运行
```

在多台机器上同时运行 TORK，它们会自动发现彼此、交换经验。
一台学到的模式，所有机器受益。
